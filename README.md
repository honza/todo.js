todo.js
=======

A fast, reliable, extensible, powerful todo list application written in
Javascript.

Compatibility
-------------

* IE
* Firefox
* Safari
* Opera
* Chrome

Features
--------

* jQuery version
* Pure javascript version
* Coffeescript version

And it even runs in [node.js](http://nodejs.org).
